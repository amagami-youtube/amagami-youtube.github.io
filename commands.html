使用できるコマンド一覧

/destroy Falseでスピーカーじゃない人をまとめて消せる
/destroy True False で全消し
/destroy blue（一般人）
/destroy darkorange（スピーカー）
/destroy red（マネージャー）
/destroy darkcyan（サミット）でそれぞれの役職を消せる
/destroy IDでIDで指定して消せる
/destroy 文字列 文字を選択して、その文字が含まれるコメントを削除（一文字でも反応するため注意）
/reduce 権限全体の2%だけを削除するcleanの代替用コマンド
/clear メッセージをすべて削除する
/del /del 1 のように番号を指定してその番号のメッセージを削除する
/NG /NG それってあなたの感想ですよね のように言葉を入力するとNGワードに登録する。
/OK /OK それってあなたの感想ですよね のように言葉を入力するとNGワードから言葉を削除する
/speaker /speaker id （のちほどidについて説明）のように入力してそのidのユーザーにメッセージの制限を無くす
/disspeaker /disspeaker id のように入力してそのidのユーザーのメッセージの制限を戻します。
/topic /topic 今の話題 を入力して今の話題を設定する
/color /color カラーコード id 運営、サミット専用のコマンド。入力して名前部分の色を決めれます。id部分は変わりません。カラーコード→https://www.colordic.org/
/manager id 運営、サミット専用のコマンド。指定した人をマネージャーにできる。
/summit id 運営専用のコマンド。指定した人をサミットにできる。
/dismanager /dismanager id ：運営、サミット専用のコマンド。入力してマネージャー権限を剝奪できる。
/dissummit /dissummit id ：運営専用のコマンド。入力してサミット権限を剝奪できる。
/add ID 文字 運営、サミット専用のコマンド。IDの最後に文字を追加可能。
/max 数字 コメントの自動削除を行う数字を指定できる。

削除されたコマンド一覧

/registe 名前、ID、シードの長さが掲示板管理部に通達されるコマンド。マネージャー以上の権限配布に使われた。 廃止日…2023年3月29日
/clean ランダムに権限持ちを剥奪するコマンド。使用されていないアカウントを削除するために使われた 廃止日…2023年3月29日
